# Télécharger OpenMandriva Lx 4.2

[Télécharger la version actuelle](/distribution/releases)

## Quelle version dois-je télécharger ?

Actuellement, le matériel le plus courant est l'architecture 64 bits des PC - Intel / AMD - également connue sous le nom de x86\_64.

D'autres architectures impliquent certainement que vous sachiez déjà ce que vous faites.

## Torrent (Peer to Peer)

BitTorrent représente une méthode simple pour télécharger OpenMandriva.

Nous avons notre propre serveur de seed à haute vitesse.

Les versions Torrent peuvent être téléchargées à partir de [archive.org](https://archive.org/details/@oma_infra). Nous avons commencé à héberger nos versions sur archive.org à partir de Lx 4.2. Les versions plus anciennes peuvent être trouvées sur [SourceForge](https://sourceforge.net/projects/openmandriva/files/release/obsolete/).

> Nous hébergeons la dernière version avec un serveur à grande vitesse pour garantir un téléchargement rapide. Les téléchargements précédents peuvent être plus lents et ne dépendent que d'archive.org et d'autres seeders disponibles.
